# Netcat

server: `nc 64.exe -l -p 4444 -e cmd.exe`\
\-l = listen -p = port -e = callback (/usr/bash/)

server alternative `nc -nvlp 4444`

client `nc server_ip port_ip`

`shutdown -i` showdown dialog for windows

`arp -a` check all ip

`msg * on99` send messageb
